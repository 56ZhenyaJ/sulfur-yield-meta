# Contributing

Contributions are welcome if they improve reproducibility, extraction clarity, or agronomic interpretation.

## Useful contributions

- Add extracted studies with clear citation keys.
- Improve sulfur source conversion notes.
- Add crop-specific sulfur-demand classifications.
- Improve variance conversion from CI, LSD, or replicate-level data.
- Add PRISMA flow data or bias-assessment templates.

## Data contribution rules

- Do not upload copyrighted PDFs.
- Do not upload publisher-owned full tables unless licensing allows it.
- Prefer extracted numeric contrast data with citation metadata.
- Mark uncertain extraction decisions in the `notes` field.

# 中文项目方案：硫肥增产效应 Meta 分析

## 研究定位

本项目聚焦“硫肥本身及硫肥配合其他元素的增产效应”。核心不是笼统回答“含硫肥料有没有增产”，而是区分以下几种效应：

1. 单施硫肥的增产效应。
2. 在相同氮肥背景下，额外添加硫的增产效应。
3. 在相同磷肥、钾肥或 NPK 背景下，额外添加硫的增产效应。
4. 硫与镁、锌、硼等中微量元素配施时的增产效应。
5. 复合施肥方案相对空白对照的“套餐效应”，仅作为敏感性分析。

## 推荐题目

中文：

> 硫肥及其养分配施对作物产量影响的 Meta 分析：基于养分背景匹配的证据综合

英文：

> Yield response to sulfur fertilizer and sulfur-based nutrient co-application in crop production: a background-matched meta-analysis

## 创新点

- 用“相同非硫养分背景下有硫 vs 无硫”替代简单的“施肥 vs 不施肥”。
- 将硫肥单施、硫氮配施、硫磷配施、硫钾配施、硫与中微量元素配施分层比较。
- 按作物需硫强度划分高、中、低响应作物。
- 纳入土壤有效硫、pH、有机质等调节变量，解释为什么有些试验增产明显、有些不明显。
- 做硫用量剂量响应，寻找边际效应下降的区间。
- 将无法拆分硫效应的“套餐施肥”单独标为敏感性证据，避免夸大硫肥贡献。

## 数据提取重点

优先提取以下对照：

| 处理 | 对照 | 解释 |
| --- | --- | --- |
| S | 无 S | 单施硫肥效应 |
| N + S | N | 氮背景下硫的边际效应 |
| P + S | P | 磷背景下硫的边际效应 |
| K + S | K | 钾背景下硫的边际效应 |
| NPK + S | NPK | 常规施肥背景下硫的边际效应 |
| NPK + S + B | NPK + B | 配施硼背景下硫的边际效应 |

## 后续投稿可讲的故事

如果数据量足够，可以把论文写成：

1. 全球/区域尺度上硫肥平均增产多少。
2. 哪类作物最值得施硫。
3. 哪些土壤条件下施硫最有效。
4. 硫与 NPK 或中微量元素配施是否比单施更稳定。
5. 多少硫用量之后增产效应开始变弱。

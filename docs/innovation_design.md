# Innovation design

## Scientific contribution

This project is designed to make a sulfur fertilizer meta-analysis more defensible than a generic fertilizer package review.

The central innovation is the contrast taxonomy:

- Sulfur marginal effect under matched nutrient background.
- Sulfur-only response under no companion nutrient background.
- Package effect kept as sensitivity evidence.

This separation supports a cleaner answer to whether sulfur itself increases yield and whether the response depends on companion nutrients.

## Analysis contribution

Planned analysis modules:

- Multilevel random-effects model with study and experiment structure.
- Treatment-role subgroup analysis.
- Crop sulfur-demand tier analysis.
- Soil available sulfur gradient.
- S-rate dose-response analysis.
- Sensitivity analysis by comparison quality and variance source.

## GitHub contribution

The repository is not just a manuscript folder. It is a reusable open workflow:

- Data extraction template.
- Field dictionary.
- Synthetic demo dataset.
- Reproducible R pipeline.
- Manuscript skeleton.
- CI check for the demo pipeline.

This makes the project easier to trust, reuse, cite, and extend.
